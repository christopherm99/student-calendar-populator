Student Calendar Populator

This website, made by Christopher Milan for a Computer Science 2 Final Project, takes the user specific PDF schedule and converts it into a .ics (iCalendar) file. This file can easily be imported into any calendar service, as iCalendar files are recognised as the standard format. PDFs are processed and converted by the websites' backend, written in Golang. The backend is hosted on GitHub.com, and can be accessed by clicking on "Backend" below. To learn more about running it yourself, check out the GitHub repository by clicking "Repository" below, where instructions are given on how to set it up.
